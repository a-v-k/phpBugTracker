2.2. Configuration Options

The options listed in this section can be configured by the configure script and can be found in config.php.

INSTALLPATH

Location on disk of the phpBugTracker installation.

INSTALLURL

URL that is the home of the installation.

CVS_WEB

Location of your cvs web interface (see format_comments() in bug.php). This is for links to files in CVS generated from the comments attached to bugs.
